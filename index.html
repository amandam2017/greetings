Greetings Exercise Description

This app allows you to enter a name and when a button is pressed display a greeting like this : Hello, . After the button is pressed the name should be removed from the text box.

    This app:
  • Greets a person with a selected language
  • It pops an error message when name and language not checked
  • It pops error message with a selected language if the required field is not filled and a language is selected
  • Allows you to view the local storage on the screen by clicking on view
